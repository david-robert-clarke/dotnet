Forwarding Mail

If you already have a mail account with an ISP or a freemail site such as hotmail you may want mail addressed to your Aston mail account to be delivered there. You may also want mail redirected elsewhere during vacation. Here's where you do it.

What are you doing and why ? When mail is delivered to your account here your home directory on pophost.aston.ac.uk is checked for a file called .forward (pronounced dot forward) which contains one or more addresses to send the mail to (normally) instead of delivering it to popmail. For example , whilst I have accounts (and many guises such as webmaster@aston.ac.uk) on all the central systems in Aston I only want to read it in one place so all those accounts have a .forward file containing 

                 oweng@aston.ac.uk
so all my mail comes to the one drop. pophost.aston.ac.uk is a Unix machine (as most Universities' heavy duty machines are) and generally folk who aren't in the IT business haven't been exposed to Unix so don't know how to go about it. And we got bored of repeatedly explaining it. Hence this form which will set the forward for you painlessly.

Enter Username (thats bloggsj NOT bloggsj.b.99)
Enter Unix/POP3 Password
Re-enter Password for verification

forward
If setting a forward enter the address in this box otherwise this field is ignored.
NOTE if the forward is not a valid mail address then mail will not get delivered but returned to sender as unknown at that address ("Bounced" in the jargon). So concentrate.
You can test whether it has worked by sending an email to your Aston address and seeing what happens.

The above only affects mail received after you have set the forward.
To download any mail previously sent to you at Aston temporarily configure your mailer thus 

    	Incoming (POP3) mailer: pophost.aston.ac.uk
    	Username & Password as setting the forward.
    	Outgoing (SMTP) mailer: not relevant.
With Outlook (which supports multiple POP3 mail servers) it goes something like this
  1. Select Accounts... from the Tools menu
  2. From Internet Accounts select Add->Mail
  3. Display Name: Not important
  4. Internet E-mail Address: Not Important
  5. E-mail Server Names
    • My incoming mail server is a POP3
    • Incoming mail server: pophost.aston.ac.uk
    • Outgoing mail(SMTP) server: not.used.so.anything
  6. Internet Mail Logon - Account name and Password as you used for setting the forward. Don't use SPA.
With Netscape

The Netscape mailer (upto 4.72 at least) only supports one POP3 server so you have to temporarily EDIT your current settings (assuming that you are using a POP3 server currently). So make a note of what you are changing so you can restore without trouble. Alternatively take a copy of your preferences file before you start - mine (Windows 98) can be found at C:\Program Files\Netscape\Users\oweng\prefs.js and (Linux/Solaris) ~/.netscape/preferences.js. Then you can restore your defaults by putting the copy back where it started.

  1. Select Preferences from the Edit menu
  2. From the Mail & Newsgroups Category; select Mail Servers
  3. Edit;
    • Server Name: pophost.aston.ac.uk
    • Server Type: POP3
    • User Name: As used to set forward
  4. From Communicator menu select Messenger
  5. Select Get Msg from toolbar
  6. Use same password as setting forward when prompted.
  7. Mail is downloaded
