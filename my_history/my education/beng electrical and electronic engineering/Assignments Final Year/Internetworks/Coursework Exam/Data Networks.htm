Data Networks Coursework

 

Course work due in: 26th January 2001

 

Describe the structure of RFC documents and explain the significance of the terms ‘must’, ‘should’ and ‘may’ in the RFC context. What relationships can exist between different RFC documents? (30%)

Find the RFC that describes the Internet Control Message Protocol (ICMP) on the Internet. Describe the method used, the location of the document and the numbers and broad content of any RFCs which are related to it. (15%)

Describe, using the current ICMP RFC, how the ICMP protocol works and include a description of the ICMP message field and a summary of the defined types of ICMP message. (15%)

Describe how ICMP can be used to create a ‘ping’ program (i.e. a program that can establish whether a host is up and running) and write pseudocode for such a program. (20%)

Choose non-zero values for the parameters required in an ICMP ECHO message (choose 32 bits of data) and compute the checksum. Change a random bit in the ICMP message and show how the checksum can be used to detect the error. Full details of the checksum calculations must be given. (10%)

Suggest an application for the ICMP timestamp message indicating briefly how it would be implemented. (10%)
